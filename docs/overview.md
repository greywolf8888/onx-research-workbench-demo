# Workbench Overview

ONX Research Workbench Demo presents a public read-only slice of a private
visual research terminal. It is intentionally small: static frontend, minimal
Python JSON service, synthetic fixtures, and documentation.

## 中文摘要

我把本仓库作为私有可视化研究终端的公开只读切片，包含静态前端、最小 Python JSON 服务、
合成样本和说明文档。
