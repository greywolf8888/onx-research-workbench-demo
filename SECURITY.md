# Security Boundary

This workbench demo is read-only and uses synthetic data.

- No order placement, cancellation, leverage edit, or account mutation exists.
- No exchange credential is required.
- No private cache or real account export is included.
- API responses are loaded from `demo_data/`.

## 中文说明

该工作台演示只读运行，不包含下单、撤单、杠杆修改或账户变更入口。
API 响应来自 `demo_data/` 合成样本。
